# Webpack Starter Kit

## 🗒️ Features

- ts, jsx support
- css, sass, scss support
- optimization for dev and production modes
- eslint (add your own rules to `.eslintrc`)

## 🚀 Quick start

1. **Clone project**

1. **Install Dependencies**

    ```shell
    npm install
    ```

1.  **Run project**

    ```shell
    npm start
    ```
